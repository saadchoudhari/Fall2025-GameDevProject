# 🎮 Fall 2025 ISU GDC Project

A Unity-based game project developed for the **Illinois State University Game Development Club (ISU GDC)**.  
This project includes multiple gameplay prototypes, power-up systems, UI elements, and scene testing environments built using Unity and C#.

---

## 🧠 Project Overview

This project serves as a collaborative prototype game environment where multiple team members developed and tested gameplay mechanics, scenes, and features.

The project includes:
- Multiple prototype scenes
- Power-up systems
- UI components
- Scene templates
- URP (Universal Render Pipeline) configuration
- TextMesh Pro integration

---

## 🛠 Tech Stack

- **Game Engine:** Unity
- **Language:** C#
- **Rendering:** Universal Render Pipeline (URP)
- **UI:** TextMesh Pro
- **Version Control:** Git
- **IDE Support:** VS Code configuration included

## 🚀 How to Open the Project

### 1️⃣ Install Unity
Make sure you have Unity Hub installed.

Open the project using the Unity version specified in:

ProjectSettings/ProjectVersion.txt


(Recommended: Use the exact Unity version listed to avoid compatibility issues.)

---

### 2️⃣ Open in Unity Hub

1. Open Unity Hub  
2. Click **Add Project**  
3. Select the `Fall2025_ISU_GDC_Project` folder  
4. Open the project  

---

## 🧪 Running the Game

1. Open any scene from:

Assets/Scenes/

2. Click the **Play** button in Unity.

Recommended starting scenes:
- `SampleScene`
- `TestScene`
- Prototype scenes inside contributor folders
